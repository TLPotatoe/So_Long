# *This project has been created as part of the 42 curriculum by tlamit.*

## Description

`so_long` is a small 2D game developed as part of the 42 school curriculum. The objective of the game is to control a character to collect all the items on the map and then move to the exit.

## Instructions

### Compilation

To compile the project, run the following command:

```bash
make
```

### Execution

To run the game, provide a map file as an argument:

```bash
./so_long maps/42.ber
```
